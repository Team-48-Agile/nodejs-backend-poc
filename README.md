# NodeJS POC for Bionic Reading using Bionic Reading API:https://rapidapi.com/bionic-reading-bionic-reading-default/api/bionic-reading1/

## Project Requirements
1. Nodejs
2. NPM

## How to run
1. npm install
2. node index.js
3. curl -v http://localhost:8084/
